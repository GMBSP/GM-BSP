wire mod has a learning curve dont worry

for a button to door setup u need to:

1. spawn a button (wire tab)
2. spawn a door or prop u want to move
3. use the wire tool to connect button's "Pressed" output to ur door
4. make sure the door has wire inputs (might need to use hydraulic or motor)

if ur just trying to move a prop, try using wire hydraulic on the prop first, THEN wire the button to the hydraulic

button -> hydraulic is easier than button -> prop directly
